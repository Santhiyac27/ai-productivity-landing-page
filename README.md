# ✨ Aura AI Landing Page

A responsive landing page for **Aura AI**, an AI-powered productivity app designed for students and professionals.

## 🚀 Features

* Responsive design (mobile + desktop)
* Dark premium UI (black + gold theme)
* Hero section with clear CTA
* Features, gallery, testimonials, and contact sections
* Smooth animations using AOS

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript

## ▶️ Run Locally

Open `index.html` in your browser.
